# Chat Application

Discord inspired chat application using Nextjs 13 and Prisma

## Features

- Create Server
- Manage Server
- Invite Friends
- Delete Server
- Leave Server
- Create Channels
- Kick Members
- Change Roles (currently Admin & Moderator) 
  ...more features are coming soon

## Requirements

- Node.js v18 & 20
- Npm v9 (Important)
